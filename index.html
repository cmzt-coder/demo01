欢迎使用我们的文档网站¶

这是一个使用 MkDocs 和 Bootswatch 主题构建的文档网站示例。

关于本项目¶

本项目演示了如何使用 MkDocs 构建专业的技术文档网站。它具有以下特点：

  • 使用 Bootswatch Cerulean 主题美化界面
  • 响应式设计，适配各种设备
  • 支持 Markdown 语法编写文档
  • 自动生成导航菜单
  • 内置代码高亮功能

文档结构¶

  • 快速开始 - 介绍如何安装和运行项目
  • 配置指南 - 详细介绍各项配置选项

测试说明¶

我们为文档网站提供了全面的测试，确保：

  1. 所有页面能够正常访问
  2. 链接没有损坏
  3. 配置文件正确加载
  4. 导航结构完整

要运行文档测试，请执行：

mkdocs serve

然后访问 http://localhost:8000 查看效果。
